1. What is the UB-04 Form?

Standard paper form for institutional claims (hospitals, rehab centers, skilled nursing facilities, etc.).

Electronic version: 837I (Institutional).

Contains detailed fields for:

Patient demographics

Facility information

Type of bill

Revenue codes

Service dates

Diagnosis and procedure codes

2. Key UB-04 Fields
Locator (FL)	Field Name	Example	Description
FL 01	Provider Name/Address	"General Hospital, 123 Main St"	Facility submitting the claim
FL 03a	Patient Control Number	"GH12345"	Internal tracking ID
FL 04	Type of Bill	"111"	1st digit: facility type, 2nd: bill classification, 3rd: frequency
FL 06	Statement Covers Period	2025-07-01 to 2025-07-05	Service date range
FL 42	Revenue Code	0300	Identifies department/service
FL 44	HCPCS/Rate	99223	Procedure code or daily rate
FL 67	Principal Diagnosis Code	I10	Main reason for admission
FL 76	Attending Physician NPI	1234567890	Physician overseeing care
3. Revenue Codes

4-digit numbers identifying the department/service for a charge.

Examples:

0300: General Medical/Surgical

0250: Pharmacy

0450: Emergency Room

0270: Medical/Surgical Supplies

4. UB-04 vs CMS-1500
UB-04 (Institutional)	CMS-1500 (Professional)
Used by hospitals, SNFs, rehab, hospice	Used by physicians, clinics
Includes revenue codes	No revenue codes
Includes type of bill code	No type of bill
837I electronic equivalent	837P electronic equivalent
